### Ejercicios de práctica (3 y 5 números)
#### Nivel básico (3 números)

* Pila a: [2, 1, 3] → Ordena de menor a mayor.
* Pila a: [3, 2, 1] → Ordena de menor a mayor.
* Pila a: [1, 3, 2] → Ordena de menor a mayor.
* Pila a: [2, 3, 1] → Ordena de menor a mayor.
* Pila a: [3, 1, 2] → Ordena de menor a mayor.

#### Nivel intermedio (5 números)

* Pila a: [3, 5, 2, 1, 4] → Ordena de menor a mayor usando pila b si es necesario.
* Pila a: [5, 1, 3, 2, 4] → Ordena de menor a mayor.
* Pila a: [2, 3, 5, 1, 4] → Ordena de menor a mayor.
* Pila a: [4, 2, 5, 3, 1] → Ordena de menor a mayor.
* Pila a: [1, 5, 3, 4, 2] → Ordena de menor a mayor.
